# vanilla-js

自分向けの Javascript ライブラリです。

| Direcoty  | Description                                      |
|-----------|-------------------------------------------------- |
| uri       | URI を処理するためのユーティリティ               |
| ajax      | fetch をラッピングしたシンプルな Ajax ライブラリ |
| redux     | シンプルかつ最小限な Redux 実装                  |
| metronome | 定期間隔での繰り返し処理用のライブラリ           |
